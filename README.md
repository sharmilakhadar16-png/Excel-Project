# Excel Sales Analysis Project

## Project Overview
This project analyzes sales data using Microsoft Excel.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Data Cleaning
- Data Visualization

## Key Features
- Cleaned and organized raw sales data
- Created Pivot Tables for performance analysis
- Calculated Revenue, Profit, and Profit Margin
- Designed charts to visualize trends

## Outcome
Identified sales trends and key performance insights using Excel analytics.
